Git is a free and open source distributed version control 
system designed to handle everything from small to very large projects 
with speed and efficiency. Git is easy to learn and has a tiny footprint 
with lightning fast performance.

What is special about Git?
One of the biggest advantages of Git is its branching capabilities. 
Unlike centralized version control systems, Git branches are cheap and easy to merge.
